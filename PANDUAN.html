🛡️1. Apa itu Patroli VP?

Patroli VP adalah aplikasi di HP untuk membantu satpam melakukan patroli dengan lebih teratur dan terdata. Semua laporan langsung tersimpan dan bisa dilihat Komandan.

💡 Aplikasi ini tidak perlu didownload dari Play Store. Cukup buka link di browser, lalu install ke homescreen HP.
👥2. Tiga Peran Pengguna
👮
Satpam

Melakukan patroli, scan QR, lapor kejadian

⭐
Komandan / Ketua RT

Setting app, lihat laporan, kelola jadwal

🏠
Ketua RW

Pantau semua RT dalam satu RW

🔑 Komandan harus setup dulu sebelum satpam bisa login. Setup dilakukan sekali saja.
📱3. Cara Install di HP Android
1
Buka browser Chrome atau Brave

Jangan gunakan browser bawaan Samsung atau browser lain. Chrome atau Brave yang paling bagus.

2
Buka link aplikasi

Ketik atau tap link yang diberikan Komandan di address bar browser. Contoh: mitrawashlaundry-code.github.io/patroli-vp

3
Tap menu titik tiga (⋮)

Ada di pojok kanan atas browser.

4
Tap "Tambahkan ke layar utama" atau "Install app"

Tap Install atau Tambah saat diminta konfirmasi.

5
Selesai! Icon muncul di homescreen

Selanjutnya buka aplikasi langsung dari icon tersebut, tidak perlu buka browser lagi.

📵 Setelah diinstall, aplikasi bisa dipakai tanpa internet (mode offline).
⭐4. Setup Pertama — Khusus Komandan

Langkah ini hanya dilakukan sekali oleh Komandan/Ketua RT saat pertama kali pakai.

01
Buka aplikasi → tap "+ Daftarkan RT Baru"

Pilih peran Komandan/Ketua RT → pilih RT Baru

02
Isi Nomor RW dan RT

Contoh: RW = 16, RT = 06

03
Isi Data RT

Nama RT, nama Komandan, nomor WhatsApp Komandan, nomor WA grup RT

04
Buat PIN

PIN Admin 4 digit (untuk Komandan) dan PIN Satpam 6 digit (dibagikan ke satpam). Catat dan simpan baik-baik!

05
Periksa data → Simpan

Tap Simpan & Mulai. Selesai! Masuk otomatis sebagai Komandan.

Setelah masuk sebagai Komandan, lanjutkan:
A
Tambah nama satpam

Tab 👥 Data Master → tap + Tambah Satpam → isi nama → Simpan. Ulangi untuk semua satpam.

B
Tambah titik patroli

Tab 👥 Data Master → tap + Tambah Titik → isi nama lokasi → Simpan. QR Code otomatis terbuat.

C
Print atau tampilkan QR Code

Tap ikon 📱 di sebelah nama lokasi → muncul QR Code → screenshot → print dan tempel di titik patroli.

D
Bagikan PIN Satpam ke satpam

Beritahu PIN 6 digit, nomor RW, dan nomor RT kepada setiap satpam.

⚠️ Jangan lupa backup! Setelah setup, masuk ke Konfigurasi → Export JSON dan simpan file backup. File ini digunakan jika HP hilang atau perlu install ulang.
🔑5. Login sebagai Satpam
1
Buka aplikasi

Tap icon Patroli VP di homescreen.

2
Isi Nomor RW dan RT

Isi sesuai yang diberitahu Komandan. Contoh: RW = 16, RT = 06.

3
Tap tombol kuning 👮 SATPAM / PETUGAS
4
Pilih nama Anda dari daftar

Tap nama Anda. Jika nama tidak ada, hubungi Komandan.

5
Masukkan PIN 6 digit

Isi kotak-kotak PIN satu per satu. Tap 🔓 Masuk.

💡 Setelah login sekali, aplikasi akan ingat Anda. Tidak perlu login lagi setiap buka aplikasi.
▶️6. Cara Mulai Shift Patroli
1
Tap tombol besar ▶ MULAI SHIFT

Ada di halaman utama setelah login.

2
Pilih nama Anda

Tap nama Anda dari daftar. Jika patroli berdua, pilih juga nama satpam kedua.

3
Tap ▶ MULAI SEKARANG

Timer langsung berjalan. Kamera otomatis aktif untuk scan QR.

4
Lakukan patroli keliling

Kunjungi setiap titik patroli dan scan QR Code yang ditempel di sana.

5
Selesai → tap ✅ SELESAI & KIRIM WA

Laporan otomatis dikirim ke WhatsApp Komandan.

⏱️ Timer tetap berjalan meski HP dikunci atau aplikasi ditutup. Tidak perlu khawatir.
📷7. Cara Scan QR Code di Titik Patroli
1
Saat shift aktif, kamera sudah otomatis hidup

Lihat bagian 📷 Scanner QR Patroli di halaman patroli.

2
Arahkan kamera ke kertas QR Code

Posisikan HP agar QR Code masuk dalam kotak panduan di layar. Jarak ideal: 15–30 cm.

3
Tunggu bunyi/getar — titik otomatis tercentang ✅

Nama titik muncul di checklist bawah dengan tanda centang hijau.

⚠️ Jika kamera tidak aktif, tap Aktifkan izin kamera yang muncul. Jika diminta izin kamera oleh HP, tap Izinkan.
🌙 Pakai fitur 🔦 Senter (tombol kecil di header) untuk scan di tempat gelap.
🚨8. Cara Lapor Kejadian Darurat
1
Tap tombol merah 🚨 LAPORKAN KEJADIAN

Bisa dilakukan kapan saja, bahkan saat tidak ada shift aktif.

2
Pilih jenis kejadian

Tap salah satu: Pencurian, Keributan, Kebakaran, Orang Asing, Kecelakaan, atau Lainnya.

3
Pilih tingkat keparahan

🟢 Ringan / 🟡 Sedang / 🔴 Berat

4
Isi deskripsi (boleh kosong)

Ceritakan singkat apa yang terjadi.

5
Tap 📤 Kirim Laporan

Laporan otomatis dikirim ke WhatsApp Komandan.

💾9. Cara Backup dan Restore Data

Backup (simpan data)

1
Login sebagai Komandan
2
Tab ⚙️ Konfigurasi → tap ⬇️ Export JSON
3
File tersimpan otomatis di folder Download HP

Namanya seperti: patroli-vp_RW16-RT06_2026-03-10.json

📅 Disarankan backup minimal seminggu sekali dan simpan file-nya di Google Drive atau WhatsApp ke diri sendiri.

Restore (setelah install ulang)

1
Buka aplikasi → di halaman login, tap tombol hijau ⬆️ Restore dari Backup JSON
2
Pilih file backup .json dari folder Download
3
Tunggu konfirmasi "✅ Restore berhasil"

RW dan RT otomatis terisi. Login seperti biasa.

❓10. Pertanyaan Umum (FAQ)
HP saya ganti baru, bagaimana data lamanya?
Selama ada file backup JSON, restore data sangat mudah. Tap tombol hijau "Restore dari Backup JSON" di halaman login, pilih file backup, selesai.
Aplikasi tidak bisa dibuka / blank putih
Coba: (1) Hapus data browser Chrome → Settings → Apps → Chrome → Clear data. (2) Buka ulang link aplikasi. (3) Install ulang dari link, lalu restore backup.
Lupa PIN Admin
PIN tersimpan di file backup JSON. Hubungi sesama Komandan yang punya akses, atau reset data dan setup ulang (data akan hilang jika tidak ada backup).
Kamera tidak bisa scan QR
Cek izin kamera: Settings HP → Apps → Brave/Chrome → Permissions → Camera → Allow. Pastikan QR Code tercetak jelas, tidak kusut, dan cahaya cukup.
Laporan WA tidak terkirim otomatis
Pastikan nomor WhatsApp Komandan sudah diisi di menu Konfigurasi → WhatsApp. Nomor harus format internasional: 628xxxxxxxx (tanpa + atau spasi).
Apakah butuh internet terus menerus?
Tidak. Setelah diinstall, aplikasi berjalan offline. Internet hanya dibutuhkan untuk kirim laporan WhatsApp dan sinkronisasi Firebase (jika diaktifkan).
Berapa banyak RT yang bisa didaftarkan?
Tidak terbatas. Setiap RT didaftarkan dengan PIN masing-masing. Ketua RW bisa memantau semua RT dalam satu RW dari satu tampilan dashboard.
